# AI Mock Interview Platform

An AI-powered mock interview web application designed to help students and job seekers practice real interview scenarios. The platform simulates technical and HR interview rounds, provides structured interview questions, and aims to generate AI-based feedback for performance improvement.

## 🚀 Features

- Mock technical and HR interview practice
- Structured interview question flow
- Answer input and response handling
- Performance feedback system (planned)
- User-friendly dashboard interface
- Future AI-based evaluation integration

## 🛠 Tech Stack

- React.js
- JavaScript
- HTML5
- CSS3
- Node.js (planned)
- Git & GitHub

## 📌 Project Status

🚧 Currently In Development

Frontend setup is completed and the core interview flow is under active development.
